  • Java JDK installed (Java 17 or Java 11)
  • Maven or Gradle installed
  • Azure Account (portal.azure.com)
  • (Optional) Azure CLI installed
  • Java App Code (e.g., Spring Boot)

ðŸ”¹ Step 1: Create Java Web Application

spring init --dependencies=web myapp
cd myapp
mvn clean package

This creates a .jar file in the target/ folder.

ðŸ”¹ Step 2: Log in to Azure Portal

Go to https://portal.azure.com

ðŸ”¹ Step 3: Create a Web App

  1. Go to App Services > + Create
  2. Choose a Resource Group and unique app name (e.g., myjavademo)
  3. Select Publish: Code
  4. Select Runtime stack: Java 17 (LTS)
  5. Select OS: Linux
  6. Select Region and App Service Plan
  7. Click Review + Create > Create

ðŸ”¹ Step 4: Deploy the Java App

Option A: Upload ZIP in Portal

  1. Compress the .jar file into a .zip
  2. Go to your App Service > Deployment Center
  3. Use the ZIP Deploy option

Option B: Use Azure CLI

az webapp deploy \
  --resource-group myResourceGroup \
  --name myjavademo \
  --src-path target/myapp.jar \
  --type jar

Option C: Use GitHub Actions

  1. Go to Deployment Center
  2. Choose GitHub as the source
  3. Select your repository and branch
  4. Azure will auto-generate a deployment workflow

ðŸ”¹ Step 5: Configure App Settings

  1. Go to App Service > Configuration
  2. Add environment variables or JVM options as needed
  3. For example, SPRING_PROFILES_ACTIVE or JAVA_OPTS

ðŸ”¹ Step 6: View Your Web App

Open in browser:

https://<your-app-name>.azurewebsites.net

To view logs:

az webapp log tail --name myjavademo --resource-group myResourceGroup
